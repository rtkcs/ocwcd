Servlet Mapping with UrlServelt

  • The container looks for matches in this order:
    1. exatch match
    2. directory match
    3. extension match
  • If the container matches more then one <url-pattern> it chooses the longest mapping.
  • Exact Mapping: /directory/urlServlet.do
  • Directory Match: /directory2/*
  • Extension Match: *.serve
